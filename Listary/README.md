## Listary Backup

Symlink to `Preferences.json` for version control.
```
mklink "D:\path\to\dotfiles\Listary\Preferences.json" "C:\Users\%USERNAME%\AppData\Roaming\Listary\UserProfile\Settings\Preferences.json"
```

```
copy /Y "C:\Users\21017\AppData\Roaming\Listary\UserProfile\Settings\Preferences.json" "D:\path\to\dotfiles\Listary\Preferences.json"
```
